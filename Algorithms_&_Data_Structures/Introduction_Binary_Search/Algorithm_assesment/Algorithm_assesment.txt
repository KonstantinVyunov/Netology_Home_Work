1. Описание алгоритма: Проходит по всем элементам и суммирует разницу текущего элемента с предыдущим.

2. Асимптотика (изменение времени работы алгоритма при существенном изменений размера входных данных):
	Памяти - констатна O(1) - вне зависимости от размера массива выделяемая память на переменные (int ans, int i) неизменна.
	Скорости обработки - линейна O(n) - необходимо перебрать все элементы массива. Чем больше элементов, тем пропорционально меньше скорость обработки.
